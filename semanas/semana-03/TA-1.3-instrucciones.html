Esta actividad de trabajo autónomo te permitirá validar la calidad de números pseudoaleatorios mediante la aplicación de pruebas estadísticas de uniformidad. Te enfocarás en aplicar pruebas de Kolmogorov-Smirnov y Chi-Cuadrado, validar medias y varianzas, y documentar resultados en una bitácora técnica.

¿Qué realizarás?

Componente Instrucciones específicas Criterios de calidad
Preparación de datos Utiliza los 1,000 números uniformes generados en PE-1.3. Verifica que estén en el rango [0,1]. Datos correctamente preparados y validados.
Prueba de Kolmogorov-Smirnov Ordena los números. Calcula desviación máxima D entre distribución empírica y teórica U(0,1). Compara con valor crítico dα,n para α=0.05 y n=1,000. Cálculo correcto del estadístico D y conclusión fundamentada.
Prueba Chi-Cuadrado Agrupa números en K=10 clases de igual amplitud. Calcula frecuencias observadas (Fo) y esperadas (Fe=100). Calcula χ² = Σ[(Fo-Fe)²/Fe]. Compara con χ²0.05,9=16.919. Tabla de frecuencias completa y estadístico χ² calculado correctamente.
Prueba de Medias Calcula media muestral x̄. Verifica hipótesis H0: μ=0.5. Calcula estadístico z = (x̄-0.5)/(σ/√n) donde σ=√(1/12). Compara con zα/2=1.96. Cálculo de media y estadístico z con interpretación correcta.
Prueba de Varianzas Calcula varianza muestral s². Verifica hipótesis H0: σ²=1/12. Calcula estadístico χ² = (n-1)s²/(1/12). Compara con valores críticos χ²0.025,999 y χ²0.975,999. Cálculo de varianza y estadístico χ² con conclusión apropiada.
Bitácora técnica Documenta cada prueba: objetivo, procedimiento, cálculos, resultados, decisión (aceptar/rechazar H0), y conclusión sobre calidad del generador. Documentación completa, clara y profesional.

Fórmulas clave

Prueba Fórmula
Kolmogorov-Smirnov D = max|Fn(x) - F(x)| donde Fn(x) es distribución empírica y F(x)=x para U(0,1)
Chi-Cuadrado χ² = Σ[(Fo - Fe)² / Fe] con K-1 grados de libertad
Prueba de Medias z = (x̄ - μ0) / (σ/√n) donde μ0=0.5, σ=√(1/12)≈0.2887
Prueba de Varianzas χ² = (n-1)s² / σ0² donde σ0²=1/12≈0.0833

Implementación en R (ejemplo)

# Cargar datos de PE-1.3
numeros <- # tus 1,000 números uniformes

# Prueba de Kolmogorov-Smirnov
ks_test <- ks.test(numeros, "punif", 0, 1)
print(paste("D =", ks_test$statistic, "p-value =", ks_test$p.value))

# Prueba Chi-Cuadrado
clases <- cut(numeros, breaks=seq(0, 1, 0.1), include.lowest=TRUE)
freq_obs <- table(clases)
freq_esp <- rep(100, 10)
chi2 <- sum((freq_obs - freq_esp)^2 / freq_esp)
print(paste("χ² =", chi2))

# Prueba de Medias
media <- mean(numeros)
sigma <- sqrt(1/12)
z <- (media - 0.5) / (sigma/sqrt(1000))
print(paste("Media =", media, "z =", z))

# Prueba de Varianzas
varianza <- var(numeros)
chi2_var <- (1000-1) * varianza / (1/12)
print(paste("Varianza =", varianza, "χ² =", chi2_var))
