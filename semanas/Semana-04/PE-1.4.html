Esta actividad práctica te permitirá aplicar el método de Monte Carlo mediante la estimación de valores numéricos. Te enfocarás en generar números aleatorios uniformes, estimar π mediante simulación, y calcular áreas bajo curvas usando integración numérica.

Ejemplo de código R:

# Estimación de π
n <- 10000
x <- runif(n)
y <- runif(n)
dentro <- sum(x^2 + y^2 <= 1)
pi_estimado <- 4 * dentro / n
error <- abs(pi_estimado - pi)
cat("π estimado:", pi_estimado, "Error:", error, "\n")

# Visualización
plot(x, y, col=ifelse(x^2+y^2<=1, "blue", "red"), 
     pch=20, cex=0.5, main="Estimación de π por Monte Carlo")

# Área bajo curva f(x) = x²
n <- 10000
x <- runif(n)
y <- runif(n)
bajo_curva <- sum(y <= x^2)
area_estimada <- bajo_curva / n
area_exacta <- 1/3
error <- abs(area_estimada - area_exacta)
cat("Área estimada:", area_estimada, "Área exacta:", area_exacta, "\n")

Fórmulas clave:

  • Estimación de π: π ≈ 4 × (puntos dentro del círculo) / (total de puntos)
  • Área bajo curva: Área ≈ (puntos bajo la curva) / (total de puntos) × (área del rectángulo)
  • Error estándar: Error ∝ 1/√n (convergencia de Monte Carlo)
